# Project Rules

## Documentation

When making changes to agents, models, memory, or pipeline logic:
- Update `README.md` if the change affects architecture, project structure, output structure, CLI flow, or the agent table
- Update `docs/brand-consulting.md` if the change affects the brand consulting pipeline (agents, workflow steps, models, memory)
- Update `docs/media-production.md` if the change affects the media production pipeline (agents, workflow steps, script generation, entity mapping)

Keep docs concise and accurate. Don't pad with filler — match the existing style.
